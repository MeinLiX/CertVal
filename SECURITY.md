## Reporting a Vulnerability

We take the security of CertVal seriously. If you discover a security vulnerability, please **do not create a public issue**, as this could expose the project and its users to attacks before a patch is ready.

### How to Report
1. **GitHub Security Advisories:** The preferred method is to use the [Private Vulnerability Reporting](https://github.com/meinlix/certval/security/advisories/new) feature on GitHub.
2. **Direct Contact:** If you cannot use GitHub, please contact the maintainers directly. 

### What to Expect
- **Acknowledgement:** We will acknowledge receipt of your vulnerability report within 48 hours.
- **Triage & Fix:** We will evaluate the issue, determine its severity, and provide a timeline for a patch.
- **Disclosure:** Once the issue is resolved and a release is published, we will publish a security advisory. We are happy to publicly credit you for the discovery if you wish.
